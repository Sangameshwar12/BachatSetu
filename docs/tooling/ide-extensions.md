# Recommended IDE Extensions

These extensions should be installed once application projects are initialized.

## VS Code

General:

- EditorConfig for VS Code
- GitLens
- GitHub Pull Requests
- YAML
- Markdown All in One
- markdownlint

Java and Spring:

- Extension Pack for Java
- Spring Boot Extension Pack
- Maven for Java

Flutter:

- Flutter
- Dart

React and TypeScript:

- ESLint
- Prettier
- TypeScript Vue Plugin is not required unless Vue is introduced
- Tailwind CSS IntelliSense only if Tailwind is selected

Infrastructure:

- HashiCorp Terraform
- Docker
- AWS Toolkit

Security and quality:

- SonarQube for IDE
- Snyk Security, if adopted by the team

## IntelliJ IDEA

Recommended edition:

- IntelliJ IDEA Ultimate for Spring Boot and database tooling

Plugins:

- Spring Boot
- Maven
- Database Tools and SQL
- Lombok, only if Lombok is approved for backend use
- CheckStyle-IDEA, if Checkstyle is adopted
- SonarQube for IDE
- GitToolBox
- AWS Toolkit
- Terraform and HCL

## Android Studio

Recommended for Flutter mobile development:

- Flutter plugin
- Dart plugin
- Android SDK tools
- Device emulator images

## Team-Level Tooling

Recommended shared tooling:

- EditorConfig
- Prettier for admin portal
- ESLint for admin portal
- Java formatter or Spotless for backend
- Flutter formatter
- markdownlint for documentation
- Dependabot for dependencies
- GitHub branch protection
