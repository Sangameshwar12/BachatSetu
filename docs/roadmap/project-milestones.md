# Project Milestones

Milestones should be used to align engineering, product, compliance, and business execution.

## Milestone 1: Repository and Architecture Approved

Exit criteria:

- Documentation reviewed
- Architecture approved
- Repository structure accepted
- Engineering workflow accepted
- First module scope confirmed

## Milestone 2: Development Foundation Ready

Exit criteria:

- Backend, mobile, and admin projects initialized
- Local development environment documented
- CI pipeline active
- Code formatting and linting active
- Database migration framework active
- Secret management approach confirmed

## Milestone 3: SaaS Core Ready

Exit criteria:

- Authentication foundation complete
- Tenant model complete
- RBAC foundation complete
- Audit logging foundation complete
- Basic admin portal access complete
- Mobile login foundation complete

## Milestone 4: Bhishi MVP Ready

Exit criteria:

- Group management complete
- Member management complete
- Contribution schedule complete
- Contribution tracking complete
- Notifications foundation complete
- Admin oversight complete

## Milestone 5: Payments and Ledger Ready

Exit criteria:

- Payment provider integration complete
- Webhook processing complete
- Idempotency complete
- Ledger model complete
- Reconciliation complete
- Payment support workflows complete

## Milestone 6: Private Beta Ready

Exit criteria:

- Staging environment production-like
- Monitoring dashboards active
- Critical tests passing
- Backup and restore tested
- Security review completed
- Support runbooks ready
- Pilot groups onboarded

## Milestone 7: Production Launch Ready

Exit criteria:

- Production environment ready
- Release pipeline approved
- Mobile app release process ready
- Security testing completed
- Performance testing completed
- Incident response process ready
- Launch checklist signed off

## Milestone 8: Scale Readiness

Exit criteria:

- Observability baseline established
- Database performance reviewed
- Background job scaling reviewed
- Cost monitoring active
- Product analytics active
- Expansion module architecture validated
